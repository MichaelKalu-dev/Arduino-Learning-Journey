# Project 00 — Blink LED

## Objective
Learn basic Arduino output control using an LED and understand program structure (setup and loop).

## Components Used
- Arduino Uno
- LED
- 220Ω resistor

## Circuit Description
The LED is connected to digital pin 7 through a resistor. The other terminal is connected to ground (GND).

## Behavior
The LED turns ON for 5 seconds and OFF for 2 seconds repeatedly in an infinite loop.

## Concepts Practiced
- pinMode() for output configuration
- digitalWrite() for controlling output states
- delay() for timing control
- Arduino program structure (setup and loop)

## Challenges Faced
- Syntax errors due to missing semicolons
- Understanding function structure in Arduino C++

## Next Step
Implement a button-controlled LED with state toggling instead of fixed timing.
